Simple Loan Calculator

This program calculates:

  • Total amount to be paid
  • Total interest
  • Monthly installment amount

Build Instructions

Compile the program using g++:

g++ -std=c++11 main.cpp -o calc

Run Instructions

Run the program with:

./calc

The program will ask for:

  • Loan amount
  • Interest rate (percentage)
  • Number of years

Example Input

Enter the loan amount: 100000 Enter the interest rate: 12 The number of years: 3

Output

It displays:

  • Total amount to be paid
  • Total interest
  • Monthly installment
